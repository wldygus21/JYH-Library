
# 개발배경

1. 본 앱은 대우정보처리회계학원 모바일포렌식을 활용한 응용 SW개발자 양성과정 수업의 산출물로서 제작한 반응형 웹입니다.
2. 사용한 언어는 PHP, MySQL, Bootstrap, jQuery를 이용하였습니다.
3. [도서관리시스템](http://jikkk8.dothome.co.kr/php) 으로 회원의 계정과 도서 관리를 할수있는 반응형 웹입니다.
